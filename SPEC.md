# 鸿蒙操作系统知识网站 - 规格说明书

## 1. 项目概述

- **项目名称**: HarmonyOS 核心技术与架构深度解析
- **项目类型**: 纯静态单页网站
- **核心功能**: 从资深程序员视角，系统性介绍鸿蒙操作系统的技术架构、核心特性与开发范式
- **目标用户**: 具有一定编程经验的开发者、架构师、技术爱好者

## 2. UI/UX 规格

### 布局结构

- **导航栏**: 固定顶部，包含章节快速跳转链接
- **Hero 区域**: 全屏高度，展示标题与核心标语
- **内容章节**: 垂直滚动布局，每个章节独立完整
- **页脚**: 版权信息与参考链接

### 响应式断点

- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 视觉设计

**色彩方案** (IDE/终端风格 - 深色主题):
- 主背景: `#0d1117` (GitHub Dark Dimmed 风格)
- 次背景: `#161b22`
- 卡片背景: `#1c2128`
- 主强调色: `#58a6ff` (代码蓝色)
- 次强调色: `#7ee787` (成功绿色)
- 警告色: `#d29922` (琥珀色)
- 错误色: `#f85149`
- 文本主色: `#e6edf3`
- 文本次色: `#8b949e`
- 代码高亮背景: `#262c36`

**字体排版**:
- 标题字体: "JetBrains Mono", "Fira Code", monospace
- 正文字体: "Noto Sans SC", "PingFang SC", sans-serif
- 代码字体: "JetBrains Mono", "Fira Code", "Consolas", monospace
- 标题大小: H1: 3rem, H2: 2rem, H3: 1.5rem
- 正文大小: 1rem (16px)
- 行高: 1.7

**间距系统**:
- 章节间距: 6rem
- 卡片内边距: 2rem
- 元素间距: 1rem

**视觉效果**:
- 卡片阴影: `0 4px 24px rgba(0, 0, 0, 0.4)`
- 边框: 1px solid `#30363d`
- 圆角: 8px
- 代码块: 左侧 4px 强调色边框
- 滚动动画: 平滑滚动 + 元素淡入

### 组件列表

1. **导航栏**: 固定定位，透明背景(滚动后加深)，包含 Logo 与章节链接
2. **Hero 区域**: 终端风格装饰，动态打字机效果标语
3. **技术卡片**: 悬停时边框发光效果
4. **代码块**: 语法高亮风格，行号显示
5. **架构图**: CSS 绘制的系统架构示意图
6. **时间线**: 鸿蒙发展历程
7. **对比表格**: 与其他系统特性对比

## 3. 功能规格

### 核心内容章节

1. **鸿蒙简介**: 定位、愿景与技术哲学
2. **微内核架构**: 与宏内核的对比、优势分析
3. **分布式架构**: 软总线、设备协同机制
4. **ArkTS 语言**: TypeScript 超集，静态类型优势
5. **Stage 模型**: 四大组件与生命周期
6. **方舟编译器**: AOT/JIT 编译策略
7. **性能优化**: 启动速度、内存管理、GPU Turbo
8. **开发工具链**: DevEco Studio 与调试技巧

### 交互功能

- 导航栏平滑滚动到对应章节
- 代码块一键复制功能
- 卡片悬停效果
- 移动端汉堡菜单
- 章节阅读进度指示器

### 数据处理

- 纯静态，无需后端
- 无需数据库
- 无需用户登录
